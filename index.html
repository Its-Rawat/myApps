Click to Download a File

Two easy ways: a direct link (for files you already have on your server/folder) and a button that generates a file on the fly.

1) Direct link (recommended)

Place your file (e.g., my-report.pdf) in the same folder as this HTML (or adjust the path). Most browsers will download it because of the download attribute.

Download Dbeaver

Change the href to your file path (e.g., files/photo.jpg) and the text label too.

2) Generate and download (JavaScript)

This creates a text file in the browser and downloads it—useful if you don’t already have a file saved on the server.

You can change the filename and contents in the script below.

Troubleshooting / tips
  • The download attribute can be ignored for cross-origin links. Keep files on the same origin for best results.
  • If your server sets Content-Disposition: attachment, browsers will download even without the attribute.
  • Binary files are fine; the browser won't corrupt them—it just saves what the server sends.
